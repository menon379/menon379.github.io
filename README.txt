# Saurabh J Menon - Product Manager Portfolio

A professional portfolio website built on the Dimension template by HTML5 UP, showcasing my experience, achievements, and skills as a Senior Product Manager with expertise in Operations, Strategy, CX, and Growth.

## About

This portfolio is built on the **Dimension** template from HTML5 UP, a fun one-pager with modal "pages" and a cool depth effect. The template is fully responsive and includes all the usual pre-styled elements.

## Features

- **Responsive Design**: Looks great on all devices
- **Interactive Elements**: Smooth navigation with depth effects
- **Professional Layout**: Clean design that highlights professional experience
- **Contact Form**: Easy way for potential employers to reach out
- **Downloadable Resume**: Direct link to my resume

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- jQuery

## Customization

This portfolio is built using the Dimension template from HTML5 UP and has been customized to include:

- Personal information and professional experience
- Key achievements with metrics
- Skills and certifications
- Contact information
- Social media links

## Sections

1. **About**: Professional summary, education, and certifications
2. **Achievements**: Key accomplishments with measurable results
3. **Experience**: Detailed work history with company links
